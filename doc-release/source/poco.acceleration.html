class PocoAccelerationMixin[source]¶

Bases: object

This class provides some high-level method to reduce redundant code implementations. As this is a MixinClass, please do not introduce new state in methods.

该mixin中定义一些常用的操作方法，将一些通用的逻辑封装起来。

dismiss(targets, exit_when=None, sleep_interval=0.5, appearance_timeout=20, timeout=120)[source]¶

自动点掉目标对象，即一直点点到全都消失为止，适用于无脑点点点的界面。

Parameters:
  • targets – <list> 目标对象列表，poco选择的对象
  • exit_when – 结束条件，默认为targets中所有节点都消失后自动退出
  • sleep_interval – 点击动作间隔，点了之后等待一段时间后再找下一个target进行点击
  • appearance_timeout – 在进行dismiss之前会等待targets中任意一个target出现，超过了这个时间还没出现就自动退出了
  • timeout – dismiss阶段超时时长
Returns:

None
Raises:

PocoTargetTimeout – 当处于dismiss切超时时，会报这个错，因为正常情况下不可能这么长时间还没把该消的消掉
